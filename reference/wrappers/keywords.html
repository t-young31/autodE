__init__(low_opt=None, grad=None, opt=None, opt_ts=None, hess=None, optts_block='', sp=None)¶

Keywords used to specify the type and method used in electronic structure theory calculations. The input file for a single point calculation will look something like:

<keyword line directive> autode.Keywords.sp[0] autode.Keywords.sp[1] autode.Keywords.optts_block

<coordinate directive> <charge> <multiplicity> . . coordinates . . <end of coordinate directive> ——————————————————————— :keyword low_opt: List of keywords for a low level optimisation :kwtype low_opt: list(str) :keyword grad: List of keywords for a gradient calculation :kwtype grad: list(str) :keyword opt: List of keywords for a low level optimisation :kwtype opt: list(str) :keyword opt_ts: List of keywords for a low level optimisation :kwtype opt_ts: list(str) :keyword hess: List of keywords for a low level optimisation :kwtype hess: list(str) :keyword optts_block: String as extra input for a TS optimisation :kwtype optts_block: str :keyword sp: List of keywords for a single point calculation :kwtype sp: list(str)
